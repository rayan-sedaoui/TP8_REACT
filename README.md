#  TP 8 : Opération Infiltration (React API Dashboard)

Bienvenue dans le dépôt officiel du **Tableau de Bord d'Espionnage v1.0**. 

Ce projet a été développé dans le cadre du TP 8 sur le Développement Front-End moderne avec React. L'objectif principal de cette mission est d'apprendre à consommer des API distantes, gérer les états de chargement et traiter les erreurs réseau.

##  Objectifs de la Mission
- Comprendre et utiliser l'API native **`fetch()`**.
- Installer et maîtriser la librairie **`axios`**.
- Gérer le cycle de vie des composants avec le Hook `useEffect`.
- Gérer les états (données, chargement, erreurs) avec le Hook `useState`.

##  Fonctionnalités du QG

Le tableau de bord est divisé en deux modules de surveillance :

1.  **InterceptionFetch** : Écoute les ondes radio et intercepte des messages codés en utilisant la méthode `fetch()`.
2.  **CiblesAxios** : Identifie les agents cibles, leurs noms de code et leurs contacts en utilisant la librairie `axios`.
